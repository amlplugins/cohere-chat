# @amlplugins/cohere-chat

Native `cohere-ai` SDK re-exported under the `@amlplugins` namespace.

> Cohere Chat — Command R/R+/A series with tools, citations, RAG-aware completions, JSON mode.

## Install

```bash
npm install @amlplugins/cohere-chat
```

## Use

```ts
import { /* named exports from cohere-ai */ } from "@amlplugins/cohere-chat";
// or
import sdk from "@amlplugins/cohere-chat";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
